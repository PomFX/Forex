📡 Auto Signal — คู่มือการใช้งาน

Smart Money Concepts (SMC) analysis framework on M15 timeframe — รองรับทุกคู่เงิน

แนะนำ

Auto Signal เป็นระบบวิเคราะห์ทางเทคนิคโดยใช้ Smart Money Concepts (SMC) บนกรอบเวลา M15 เพื่อวิเคราะห์และสร้างสัญญาณเทรดอัตโนมัติ รองรับการวิเคราะห์หลายคู่เงินในครั้งเดียว

Key Features
  • ครอบคลุมทุกคู่: Commodities (ทอง, เงิน), Forex (คู่หลัก), Crypto (BTC, ETH, XRP)
  • SMC M15: วิเคราะห์ Market Structure, Order Block, FVG, Liquidity
  • เลือกคู่: เลือกเฉพาะคู่ที่ต้องการ เปิด/ปิด ได้ทุกคู่
  • Manual Mode: วิเคราะห์เอง → ตรวจสอบ → ยืนยันส่ง
  • Auto Mode: ตั้งเวลาปล่อยให้รันอัตโนมัติ

การเข้าใช้งาน

  1. เข้าสู่ระบบด้วยบัญชี Admin
  2. คลิกที่เมนู Auto Signal ในแถบเมนูของ Admin Panel
  3. หรือพิมพ์ URL: /#/admin/autosignal

เลือกคู่เงิน

ในหน้า Auto Signal จะมีรายการคู่เงินแบ่งตามหมวดหมู่:

หมวดคู่เงินรายละเอียด
🏆 CommoditiesXAU/USD, XAG/USDทองคำและเงิน
💱 ForexEUR/USD, GBP/USD, USD/JPY, USD/CHF, AUD/USD, NZD/USD, USD/CADคู่เงินหลักทั้งหมด
🪙 CryptoBTC/USD, ETH/USD, XRP/USDสกุลเงินดิจิทัล

Toggle เปิด/ปิด ได้ตามต้องการ — คู่ที่เปิด ✅ ส่ง จะถูกนำไปวิเคราะห์ คู่ที่ปิด ⛔ ไม่ส่ง จะถูกข้ามไป

SMC (Smart Money Concepts) คืออะไร

SMC เป็นแนวคิดการวิเคราะห์ทางเทคนิคที่มองว่าราคามีการเคลื่อนไหวตาม "เงินใหญ่" (Smart Money) — สถาบันการเงิน นักลงทุนรายใหญ่ ฯลฯ

5 องค์ประกอบหลักที่ใช้วิเคราะห์

องค์ประกอบคำอธิบาย
1. Market Structureโครงสร้างตลาด — HH/HL (แนวโน้มขาขึ้น), LH/LL (แนวโน้มขาลง), CHoCH (เปลี่ยนแนวโน้ม), BOS (ทะลุโครงสร้าง)
2. Order Block (OB)โซนที่ราคามีคำสั่งซื้อ/ขายขนาดใหญ่สะสม — Bullish OB / Bearish OB
3. Fair Value Gap (FVG)ช่องว่างของราคาที่เกิดจาก 3 แท่งเทียน — แสดงถึงความไม่สมดุลของแรงซื้อแรงขาย
4. Liquidityโซนสภาพคล่อง — จุดที่ราคามักจะไป "เก็บสภาพคล่อง" ก่อนกลับตัว (Stop Hunt)
5. Entry Conditionต้องครบทั้ง 2 เงื่อนไข: (A) โครงสร้างตลาดเปลี่ยน (BOS/CHoCH) และ (B) ราคาอยู่ที่ OB/FVG + เก็บสภาพคล่องแล้ว

วิเคราะห์ SMC

เมื่อเลือกคู่เงินที่ต้องการแล้ว กดปุ่ม 🔍 วิเคราะห์ SMC

ขั้นตอนการทำงาน
  1. ระบบบันทึกการตั้งค่าปัจจุบัน (คู่ที่เลือก, auto mode, interval)
  2. เรียก OpenAI gpt-4o-mini เพื่อวิเคราะห์ SMC ทีละคู่
  3. AI วิเคราะห์ Market Structure, OB, FVG, Liquidity บน M15
  4. แสดงผลลัพธ์ — คู่ที่มี Setup และคู่ที่ไม่มี
  5. รอให้ Admin ยืนยันหรือยกเลิก

การวิเคราะห์ 1 คู่ใช้เวลา ~5-15 วินาที ยิ่งหลายคู่ยิ่งนานตามลำดับ

อ่านผลลัพธ์

ผลการวิเคราะห์จะแสดงเป็นกล่องๆ แยกตามคู่เงิน:

🟢 มี SMC Setup — แสดง:

  • คู่เงิน + ทิศทาง (BUY/SELL)
  • ราคาเข้า (Entry), Take Profit 1-3, Stop Loss
  • เหตุผลทางเทคนิค 3 บรรทัด (ภาษาไทย)

⚫ ไม่มี SMC Setup — ระบบจะบอกว่าคู่นั้นไม่มี Setup ในขณะนี้

🔴 Error — ถ้ามีข้อผิดพลาด (เช่น OpenAI Key ไม่ถูกต้อง) จะแสดง Error พร้อมรายละเอียด

ส่งแบบ Manual

ขั้นตอนการส่ง Signal แบบ Manual (ปิด Auto Mode):

  1. เลือกคู่เงินที่ต้องการ → Toggle ON
  2. กด 🔍 วิเคราะห์ SMC
  3. รอผลการวิเคราะห์ (~5-15 วิ/คู่)
  4. ตรวจสอบผลลัพธ์
  5. กด ✅ ยืนยันส่ง Signal เพื่อส่ง หรือ ❌ ยกเลิก เพื่อยกเลิก
💡 Tips: หลังยืนยันส่ง Signal จะเข้าไปอยู่ในระบบทันที MT5 EA จะดึงไปเปิด Pending Order ภายใน 60 วินาที

ส่งแบบ Auto

การตั้งค่าให้ระบบทำงานอัตโนมัติ:

  1. เลือกคู่เงินที่ต้องการวิเคราะห์
  2. เปิด Switch Auto Mode → ON
  3. เลือก Interval ที่ต้องการ
  4. กด 💾 บันทึกการตั้งค่า

การทำงาน

  • GitHub Actions เรียก POST /api/auto-signals/auto-run ทุก 15 นาที
  • Server ตรวจสอบว่า Auto Mode ON หรือไม่
  • เช็คว่าครบกำหนด Interval หรือยัง (จับเวลาจาก lastRunAt)
  • ถ้าครบ → วิเคราะห์ SMC ทุกคู่ที่เปิด → ส่ง Signal อัตโนมัติ → อัปเดต lastRunAt
  • ถ้ายังไม่ครบ → ข้ามไป รอรอบถัดไป

ยืนยัน / ยกเลิก

ใน Manual Mode หลังจากวิเคราะห์เสร็จ จะมีปุ่มให้เลือก:

ปุ่มการทำงาน
✅ ยืนยันส่ง Signalส่ง Signal ทั้งหมดที่มี Setup เข้าสู่ระบบ → EA ดึงไปเปิด Pending Order
❌ ยกเลิกยกเลิกผลการวิเคราะห์ครั้งนี้ ไม่ส่ง Signal ใดๆ
⚠️ หมายเหตุ: Auto Mode จะ ส่งอัตโนมัติทันที โดยไม่ต้องยืนยัน — ไม่มีขั้นตอนตรวจสอบก่อนส่ง

GitHub Actions

ระบบ Auto Signal ใช้ GitHub Actions เพื่อเรียก API เป็นระยะ

รายการค่า

Secrets ที่ต้องตั้งใน GitHub

AI_API_URL = https://forex-rouge-gamma.vercel.app
AI_SIGNAL_API_KEY = (ค่าจากใน .env)

ตรวจสอบสถานะ

วิธีตรวจสอบว่าระบบทำงานปกติ:

  • GitHub Actions: ไปที่ GitHub → Actions → Auto Signal Runner → ดูว่าล่าสุด Success หรือ Failed
  • Admin Panel → Signals: ดูว่ามี Signal ใหม่เกิดขึ้นตาม Interval หรือไม่
  • MT5 Expert Log: ดูว่า EA ดึง Signal มาวาง Pending Order หรือไม่
  • test-signal.html: ใช้หน้า test-signal.html ดู Signal ที่ Active

แก้ปัญหาเบื้องต้น

❌ วิเคราะห์ไม่ได้ "OPENAI_API_KEY not set on server"

ไปตั้ง OPENAI_API_KEY ที่ Vercel Dashboard → Settings → Environment Variables

❌ GitHub Actions Failed

เช็คว่า AI_API_URL และ AI_SIGNAL_API_KEY อยู่ใน GitHub Secrets หรือไม่

❌ Auto Mode ไม่ส่ง Signal

เช็ค 3 อย่าง: (1) Auto Mode ON? (2) มีคู่ที่เลือกหรือไม่? (3) Interval ครบกำหนดหรือยัง?

❌ วิเคราะห์แล้วไม่พบ SMC Setup

เป็นเรื่องปกติ — SMC จะสร้างสัญญาณเมื่อเงื่อนไขครบเท่านั้น ถ้าตลาดไม่มีโครงสร้างชัดเจน AI จะไม่สร้างสัญญาณ

❌ MT5 ไม่เปิด Pending Order

เช็ค: (1) EA อยู่บน Chart หรือไม่ (2) WebRequest URL ตั้งไว้หรือยัง (3) สัญญาณ status = active

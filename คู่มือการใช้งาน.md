## 1. ภาพรวมระบบ

ATH Trader เป็นเว็บไซต์ระบบจัดการสัญญาณเทรด Forex แบบครบวงจร ประกอบด้วย:

- **หน้าเว็บสำหรับผู้ใช้งานทั่วไป** — ดูสัญญาณเทรด บทความ โบรกเกอร์ ติดต่อ
- **แผงผู้ดูแลระบบ (Admin)** — จัดการข้อมูลทั้งหมดผ่านเว็บ

**เทคโนโลยีที่ใช้:**
- Frontend: HTML, CSS, JavaScript (Single Page Application)
- Backend: Node.js + Express
- Database: PostgreSQL (Neon)
- Hosting: Vercel

## 3. หน้าแรก

หน้าแรกแสดงข้อมูลสำคัญดังนี้:

| ส่วน | รายละเอียด |
|------|------------|
| สถิติ | สัญญาณทั้งหมด, BUY ที่ชนะ, SELL ที่ชนะ, สมาชิก VIP |
| ช่องทางติดต่อ | QR Code Line, OpenChat, TikTok (เรียงแนวนอน) |
| สัญญาณเทรดล่าสุด | 3 สัญญาณล่าสุด (คลิก "ดูทั้งหมด" เพื่อดูเพิ่ม) |
| บทความล่าสุด | 3 บทความล่าสุด (คลิก "อ่านทั้งหมด" เพื่อดูเพิ่ม) |

---

## 4. สัญญาณเทรด

แสดงรายการสัญญาณเทรดทั้งหมด เรียงตามวันที่ล่าสุด

**ข้อมูลในแต่ละสัญญาณ:**
- คู่เงิน (เช่น EUR/USD, XAU/USD)
- ทิศทาง (BUY / SELL)
- ราคาเข้า / TP1-3 / SL
- สถานะ (win / loss / pending)
- วันที่

---

## 5. บทความ

แสดงบทความและข่าวสาร การ์ดแต่ละใบแสดง:
- รูปภาพประกอบ
- หัวข้อ
- เนื้อหา (แสดงเต็ม ไม่มีการตัดคำ)
- วันที่

---

## 6. โบรกเกอร์

รายการโบรกเกอร์แนะนำ พร้อม:
- ชื่อโบรกเกอร์
- คะแนน (1-5)
- คำอธิบาย
- ลิงก์ IB สำหรับสมัคร
- โลโก้

---

## 7. แพ็กเกจ VIP

แพ็กเกจสมาชิก:
- **Free** — เริ่มต้นฟรี
- **Basic** — สัญญาณพื้นฐาน
- **Premium** — สัญญาณพรีเมียม
- **VIP** — สัญญาณ VIP + วิเคราะห์ส่วนตัว

## 8. ติดต่อ

ช่องทางการติดต่อทั้งหมด (คลิกที่การ์ดเพื่อเปิดลิงก์):
- Line ID / Line Official (QR Code)
- โทรศัพท์
- อีเมล
- Facebook
- TikTok (QR Code)
- YouTube
- OpenChat (QR Code)

## 9. การสมัครสมาชิก / เข้าสู่ระบบ

### สมัครสมาชิก
1. คลิกเมนู "เข้าสู่ระบบ"
2. เลือก "สมัครสมาชิก"
3. กรอกชื่อผู้ใช้, อีเมล, รหัสผ่าน
4. กดสมัคร

### เข้าสู่ระบบ
1. คลิกเมนู "เข้าสู่ระบบ"
2. กรอกอีเมลและรหัสผ่าน
3. ถ้าเป็นแอดมิน จะเข้าสู่หน้า Admin อัตโนมัติ

## 10. แผงผู้ดูแลระบบ (Admin)

**การเข้า Admin:**
- เมนู → เข้าสู่ระบบ
- อีเมล: `admin@athtrader.com`
- รหัสผ่าน: `admin123`
- หรือเข้า URL: `https://forex-rouge-gamma.vercel.app/#/admin`

### 10.1 Dashboard
ภาพรวมระบบ:
- สมาชิกทั้งหมด
- สมาชิก VIP
- สัญญาณทั้งหมด
- บทความทั้งหมด
- โบรกเกอร์ทั้งหมด

### 10.2 จัดการสมาชิก
- ดูรายชื่อสมาชิกทั้งหมด
- เปลี่ยนระดับ VIP (Free / Basic / Premium / VIP)
- ลบสมาชิก

### 10.3 จัดการสัญญาณเทรด
- เพิ่มสัญญาณใหม่: กรอกคู่เงิน, ทิศทาง, ราคา, TP1-3, SL
- แก้ไขสัญญาณ
- ลบสัญญาณ
- เปลี่ยนสถานะ (win / loss / pending)

### 10.4 จัดการบทความ
- เพิ่มบทความ: หัวข้อ, เนื้อหา, รูปภาพ
- แก้ไขบทความ
- ลบบทความ
- อัปโหลดรูปภาพประกอบ

### 10.5 จัดการโบรกเกอร์
- เพิ่มโบรกเกอร์: ชื่อ, คะแนน, คำอธิบาย, ลิงก์ IB, โลโก้
- แก้ไขโบรกเกอร์
- ลบโบรกเกอร์

### 10.6 จัดการช่องทางติดต่อ
- **Line ID** — กรอก ID หรือลิงก์ Line
- **โทรศัพท์** — เบอร์โทรศัพท์
- **อีเมล** — ที่อยู่อีเมล
- **Facebook** — ลิงก์เพจ Facebook
- **TikTok** — ลิงก์ TikTok
- **YouTube** — ลิงก์ YouTube
- **OpenChat** — ลิงก์เข้ากลุ่ม OpenChat
- **QR Code Line** — อัปโหลดหรือวางลิงก์รูป QR
- **QR Code OpenChat** — อัปโหลดหรือวางลิงก์รูป QR
- **QR Code TikTok** — อัปโหลดหรือวางลิงก์รูป QR

> **หมายเหตุ:** กดปุ่ม "บันทึก" ทุกครั้งหลังจากแก้ไขข้อมูล

## 11. การอัปโหลดรูปภาพ

ระบบใช้ **Base64 Data URL** สำหรับเก็บรูปภาพ (รองรับ Vercel serverless):

1. กดปุ่ม "เลือกไฟล์"
2. เลือกรูปภาพจากเครื่อง
3. รอข้อความ "✔ อัปโหลดแล้ว"
4. ระบบจะใส่ลิงก์ Base64 ให้อัตโนมัติ
5. กด "บันทึก" เพื่อจัดเก็บ

## 12. การปรับแต่งเบื้องต้น

### เปลี่ยนสีธีม
แก้ไขใน `css/style.css`:
```css
:root {
  --gold: #FFD700;
  --gold-dark: #DAA520;
  /* ปรับค่าสีอื่นๆ ได้ */
}
```

### เปลี่ยนชื่อเว็บ
ค้นหา `ATH Trader` ใน `index.html` แล้วเปลี่ยนตามต้องการ

### ค่าตัวเลขสถิติหน้าแรก
ใน `server/routes/stats.js`:
```js
buyWins: 10,
sellWins: 30,
vipCount: 1250,
```

### การ Deploy
```bash
npm run dev          # รัน localhost:8080
vercel deploy --prod # อัปเดตขึ้น production
```

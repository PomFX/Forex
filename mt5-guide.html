📡 คู่มือการใช้งาน EA SignalReceiver

รับสัญญาณเทรดอัตโนมัติ — ATH Trader MT5 Integration

แนะนำ EA

SignalReceiver เป็น Expert Advisor (EA) สำหรับ MetaTrader 5 ที่พัฒนาโดย ATH Trader เพื่อรับสัญญาณเทรด (Trading Signals) จากระบบของเราแล้วเปิด Pending Order อัตโนมัติ

คุณสมบัติ
  • รับสัญญาณจาก API อัตโนมัติทุกๆ 60 วินาที
  • รองรับ Symbol หลากหลาย: XAU/USD, Forex, Crypto
  • เลือกรับเฉพาะหมวดที่ต้องการ (Commodities/Forex/Crypto)
  • เลือก TP Mode (TP1/TP2/TP3)
  • คำนวณ Lot ตามความเสี่ยง (Risk %)
  • ตรวจจับ Symbol อัตโนมัติ ไม่ว่าจะชื่ออะไร (.m, .pro, GOLD ฯลฯ)
  • ยกเลิก Pending Order เก่าเมื่อมีสัญญาณใหม่
  • ปุ่มลัด F5 = ตรวจสัญญาณทันที, F6 = ยกเลิกทั้งหมด

สิ่งที่ต้องเตรียม

รายการรายละเอียด
MetaTrader 5ติดตั้ง MT5 จาก Broker ที่คุณใช้ (หรือ MT5 ทั่วไป)
Expert Advisorsต้องเปิดใช้งาน EA: Tools → Options → Expert Advisors → Allow Automated Trading
WebRequestอนุญาตให้ EA เชื่อมต่อ API ได้ (ตั้งค่าด้านล่าง)
อินเทอร์เน็ตต้องการการเชื่อมต่อที่เสถียร (แนะนำ VPS)
สิทธิ์ Adminแนะนำ Run as Administrator เพื่อป้องกัน Error 5203

ติดตั้ง EA

1

ดาวน์โหลดไฟล์ EA

ดาวน์โหลดไฟล์ SignalReceiver.mq5 จากลิงก์ที่ Admin จัดเตรียมไว้

2

เปิด MT5 Data Folder

ใน MT5 → File → Open Data Folder → ไปที่ MQL5 → Experts

3

วางไฟล์ EA

วางไฟล์ SignalReceiver.mq5 ในโฟลเดอร์ MQL5/Experts/

4

Compile EA

เปิด MetaEditor (F4) หรือใน MT5 → Tools → MetaQuotes Language Editor → เปิดไฟล์ → กด F7 (Compile)

5

ตั้งค่า WebRequest

Tools → Options → Expert Advisors → ติ๊ก Allow WebRequest + เพิ่ม URL (ดูหัวข้อถัดไป)

แนบ EA เข้า Chart

1

เปิด Chart ที่ต้องการ (เช่น XAU/USD, GOLD, ETHUSDi)

2

ลาก EA SignalReceiver จาก Navigator (View → Navigator) วางบน Chart

3

หน้าต่าง Setting จะเปิดขึ้นมา → ตั้งค่าตามต้องการ → กด OK

⚠️ ข้อสำคัญ: ถ้าเป็นครั้งแรก ต้องติ๊ก "Allow Automated Trading" ในหน้าต่าง Common ก่อน

EA จะเริ่มทำงานทันที — จะเห็นข้อความขึ้นใน Experts log ว่า [SignalReceiver v2.2] MULTI-SYMBOL PENDING ORDER EA

ตั้งค่า Inputs (พารามิเตอร์)

เมื่อแนบ EA เข้า Chart จะมีหน้าต่าง Inputs ให้ตั้งค่า:

พารามิเตอร์ค่าเริ่มต้นคำอธิบาย
API_URLhttps://forex-rouge-gamma.vercel.app/api/signals/mt5URL สำหรับดึงสัญญาณ ไม่ต้องแก้
API_KEYd0d52f...API Key สำหรับยืนยันตัวตน ไม่ต้องแก้
LOT_SIZE0.01ขนาด Lot คงที่ (ใช้เมื่อ USE_RISK_PCT = false)
POLL_INTERVAL60ความถี่ในการตรวจสอบสัญญาณ (วินาที)
MAGIC_NUMBER20260605Magic Number ใช้แยกคำสั่งของ EA นี้
USE_RISK_PCTfalsetrue = คำนวณ Lot ตามความเสี่ยง, false = ใช้ LOT_SIZE คงที่
RISK_PERCENT1.0ความเสี่ยงต่อการเทรด (%) — ใช้เมื่อ USE_RISK_PCT = true
TP_MODE11=TP1, 2=TP2, 3=TP3
FILTER_COMMODITIEStrueรับสัญญาณ Commodities (XAU/USD, XAG/USD)
FILTER_FOREXtrueรับสัญญาณ Forex (EUR, GBP, JPY, CHF, AUD, NZD, CAD)
FILTER_CRYPTOtrueรับสัญญาณ Crypto (BTC, ETH, XRP)
💡 Tips: ถ้าวาง EA บน Chart XAU/USD และเปิด FILTER_COMMODITIES = true, FILTER_FOREX = false, FILTER_CRYPTO = false — EA จะรับเฉพาะสัญญาณทองคำเท่านั้น

Symbol ที่รองรับ

EA สามารถตรวจจับชื่อ Symbol ได้อัตโนมัติ ไม่ว่า Broker จะตั้งชื่ออะไร:

ประเภทชื่อในระบบชื่อใน Broker (ตัวอย่าง)
ทองคำXAU/USDXAUUSD, GOLD, XAUUSD.m, XAUUSD.pro, XAUUSD.r, XAUUSD.ecn, XAUUSDu
เงินXAG/USDXAGUSD, SILVER, XAGUSD.m
ForexEUR/USD, GBP/USD, ฯลฯEURUSD, EURUSD.m, EURUSD.pro, EURUSDu
CryptoBTC/USD, ETH/USD, XRP/USDBTCUSD, BTCUSD.m, ETHUSDi, XRPUSDi
อื่นๆกรอกเองค้นหาอัตโนมัติจากทุกรายการ Symbol ใน Broker

EA จะค้นหา Symbol โดยอัตโนมัติ: เปลี่ยน / เป็นว่าง → ลองชื่อตรง → ลอง GOLD → ลอง suffix ต่างๆ → ค้นหาจากทุกรายการที่ Broker มี

ระบบทำงานอย่างไร

  1. API — ระบบ ATH Trader สร้างสัญญาณเทรด (Auto Signal หรือ Manual)
  2. EA — เรียก API ทุกๆ POLL_INTERVAL วินาที (ค่าเริ่มต้น 60 วิ)
  3. รับข้อมูล — EA ได้รับ JSON: {"id":...,"pair":"XAU/USD","direction":"BUY","entry":4318.50,...}
  4. กรอง — เช็ค Category Filter + เปรียบเทียบ Symbol กับ Chart ปัจจุบัน
  5. ค้นหา Symbol — FindSymbol() เปลี่ยนชื่อ API pair → MT5 symbol
  6. ยกเลิกคำสั่งเก่า — ลบ Pending Order ก่อนหน้าทั้งหมด (Magic Number เดียวกัน)
  7. เปิดคำสั่งใหม่ — เลือกประเภท LIMIT/STOP อัตโนมัติตาม Entry vs ราคาตลาด
🔄 วนรอบ: ทุกๆ 60 วิ EA จะตรวจสอบสัญญาณใหม่ ถ้าเจอสัญญาณใหม่ → แทนที่อันเก่า ถ้าเจอ null → ยกเลิกทั้งหมด

Pending Order (คำสั่งตั้งรอ)

EA จะเลือกประเภท Pending Order อัตโนมัติ:

ทิศทางEntry vs ราคาประเภทคำอธิบาย
BUYEntry < ราคาปัจจุบันBUY LIMITรอราคาลงมาที่ Entry
BUYEntry > ราคาปัจจุบันBUY STOPรอราคาขึ้นไปที่ Entry
SELLEntry > ราคาปัจจุบันSELL LIMITรอราคาขึ้นไปที่ Entry
SELLEntry < ราคาปัจจุบันSELL STOPรอราคาลงมาที่ Entry

Pending Order จะถูกยกเลิกเมื่อ:

  • มีสัญญาณใหม่มาแทนที่ (Signal ID ใหม่กว่า)
  • สัญญาณปัจจุบันถูกเปลี่ยนสถานะเป็น win/loss
  • กด F6 หรือกด Cancel ใน test-signal.html

ปุ่มลัด (Hotkeys)

เมื่อ EA ทำงานอยู่บน Chart สามารถใช้ปุ่มลัดได้:

ปุ่มการทำงาน
F5ตรวจสอบสัญญาณทันที (ไม่ต้องรอรอบถัดไป)
F6ยกเลิก Pending Order ทั้งหมดของ EA นี้

อ่าน Log (Expert Log)

ดู Log ได้ที่ View → Toolbox → Experts หรือกด F8

ข้อความความหมาย
[SignalReceiver v2.2] MULTI-SYMBOL PENDING ORDER EAEA เริ่มทำงานสำเร็จ
[SignalReceiver] Checking for new signal...กำลังตรวจสอบ API

[SignalReceiver] WebRequest FAILED...เชื่อมต่อ API ไม่ได้ (ดูหัวข้อแก้ปัญหา)

ตั้งค่า WebRequest

EA ใช้ WebRequest เพื่อเชื่อมต่อ API ต้องตั้งค่าก่อนใช้งาน:

1

เปิด MT5 → Tools → Options → Expert Advisors

2

ติ๊ก ✅ Allow WebRequest for listed URL

3

เพิ่ม URL นี้ในลิสต์:

https://forex-rouge-gamma.vercel.app

หรือถ้ายัง Error 5203 ให้ใส่ path เต็ม:

https://forex-rouge-gamma.vercel.app/api/signals/mt5
4

กด OK → ปิด MT5 แล้วเปิดใหม่ (Restart)

⚠️ ถ้ายัง Error 5203: ให้คลิกขวา MT5 → Run as Administrator → เปิด Chart → Attach EA ใหม่

Run as Administrator

Windows Server / VPS มักปิดกั้น WinHTTP สำหรับโปรแกรมที่ไม่ได้รันด้วยสิทธิ์ Admin → MT5 ใช้ WinHTTP ในการทำ WebRequest → ถ้า Error 5203 ต้องรัน MT5 แบบ Administrator

วิธีทำ:

  1. ปิด MT5
  2. คลิกขวาที่ไอคอน MT5 → Run as Administrator
  3. กด Yes เพื่อยืนยัน
  4. เปิด Chart → Attach EA

ใช้งานบน VPS

แนะนำให้ใช้ VPS (Virtual Private Server) เพื่อให้ EA ทำงาน 24/7 โดยไม่ต้องเปิดเครื่องทิ้งไว้

💡 Tips สำหรับ VPS:
  • เข้า VPS ผ่าน Remote Desktop (RDP)
  • เปิด MT5 → ตั้งค่า WebRequest → Run as Administrator → Attach EA
  • ตั้ง Windows ให้ล็อกอินอัตโนมัติ และ MT5 เปิดอัตโนมัติเมื่อรีสตาร์ท
  • ใช้ไฟล์ WebRequestTest.mq5 ทดสอบการเชื่อมต่อ API ก่อนใช้ EA จริง

ปัญหาที่พบบ่อย

❌ WebRequest FAILED | res=1001 LastError=5203

สาเหตุ: URL ไม่อยู่ใน Allowed List หรือ MT5 ไม่มีสิทธิ์เข้าถึงเครือข่าย

แก้ไข:

  • เช็ค Tools → Options → Expert Advisors → Allow WebRequest → มี URL หรือยัง
  • ใส่ path เต็ม: https://forex-rouge-gamma.vercel.app/api/signals/mt5
  • Run as Administrator → เปิด MT5 ใหม่
  • ปิด Windows Firewall ชั่วคราวเพื่อทดสอบ

❌ Pending order FAILED | RetCode=10018

สาเหตุ: ตลาดปิด (Market closed) — เสาร์-อาทิตย์ หรือ Symbol นั้นไม่เปิดให้เทรด

แก้ไข: รอตลาดเปิด หรือตรวจสอบ Trading Hours ของ Symbol นั้นใน Market Watch → Specification

❌ Pending order FAILED | RetCode=10014

สาเหตุ: ราคาที่ขอไม่ถูกต้อง (Invalid price) — Entry/SL/TP ไม่ตรงกับ Stop Level

แก้ไข: ระบบปรับ SL/TP อัตโนมัติให้ถูกต้อง ลองใหม่อีกครั้งเมื่อราคาเปลี่ยน

❌ Symbol not found for: XXX

สาเหตุ: Broker ของคุณไม่มี Symbol นี้ หรือใช้ชื่ออื่น

แก้ไข: เพิ่มชื่อ Symbol ใน Market Watch (คลิกขวา → Symbols → แสดง Symbol นั้น)

❌ EA ไม่ทำงาน — Expert หยุด

สาเหตุ: เปิด MT5 ครั้งแรก โดยไม่ได้ตั้งค่า Allow Automated Trading

แก้ไข: ใน Chart → คลิก笑脸 → เปลี่ยน Enable เป็น On หรือกดปุ่ม AutoTrading

❌ Trade context busy

สาเหตุ: MT5 กำลังทำงานอื่นอยู่ (เช่น สั่งซื้อพร้อมกันหลายอัน)

แก้ไข: EA จะลองอีกครั้งในรอบถัดไป (60 วิ) — ปกติไม่ต้องทำอะไร

Error Codes (RetCode)

Codeความหมาย
10004Trade disabled (ปิด AutoTrading)
10006No connection (MT5 ไม่ต่อ Server)
10014Invalid price (ราคาไม่ถูกต้อง)
10018Market closed (ตลาดปิด)
10019Too many requests (คำสั่งมากเกินไป)
130Invalid stops (SL/TP ห่างจากราคาน้อยเกินไป)
138Requote (ราคาเปลี่ยนระหว่างส่งคำสั่ง)
145Modification denied (แก้ไขไม่ได้รับอนุญาต)

ติดต่อช่วยเหลือ

  • ติดต่อ Admin ผ่านช่องทางที่กำหนด
  • แจ้ง Error Code และ Log เพื่อให้วิเคราะห์ปัญหาได้เร็วขึ้น

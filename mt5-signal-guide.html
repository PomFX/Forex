คู่มือระบบส่ง Signal MT5 → LINE

ระบบรับสัญญาณ BOS/CHoCH จาก LuxAlgo SMC บน MetaTrader 5 ประมวลผลด้วย AI แล้วส่งแจ้งเตือนเข้ากลุ่ม/ผู้ใช้ LINE อัตโนมัติ

🔄 Flow การทำงาน

ขั้นตอนทั้งหมด
  1. MT5 ตรวจจับ BOS/CHoCH จาก LuxAlgo SMC indicator บน chart
  2. EA ส่งข้อมูล ผ่าน HTTP POST ไปยัง Backend ของ ATH Trader
  3. Backend คำนวณ Entry / SL / TP1-3 ตามกลยุทธ์ BOS + Order Block
  4. AI วิเคราะห์ Confidence และ Risk:Reward (ถ้าเปิดใช้งาน)
  5. บันทึกลง Database เป็นสัญญาณเทรด
  6. ส่งแจ้งเตือน LINE ตามเป้าหมายและ Plan ที่ตั้งไว้

⚙️ Vercel Environment Variables

ตั้งค่าตัวแปรเหล่านี้ใน Vercel Dashboard → Settings → Environment Variables

ตัวแปรค่าที่ใส่จำเป็น
DATABASE_URLPostgreSQL connection stringจำเป็น
JWT_SECRETSecret สำหรับ sign tokenจำเป็น
ADMIN_EMAILEmail Super Adminจำเป็น
ADMIN_PASSWORDPassword Super Adminจำเป็น
MT5_API_KEYAPI key สำหรับ MT5 bridgeจำเป็น
OPENAI_API_KEYOpenAI API keyถ้าใช้ AI
LINE_CHANNEL_ACCESS_TOKENToken จาก LINE OAจำเป็น
LINE_CHANNEL_SECRETSecret จาก LINE OAแนะนำ
LINE_USER_IDUser ID ของคุณแนะนำ
⚠️ สำคัญ: ทุกครั้งที่แก้ Environment Variables ต้องกด Save แล้ว Redeploy เพื่อให้ค่ามีผล

🖥️ หน้า MT5 Signal Settings

เข้าถึงที่: /#/admin/mt5signals (ต้อง login เป็น Admin ก่อน)

ตัวเลือกหลัก
  • Admin อนุมัติก่อนส่ง LINE — Signal จาก MT5 จะบันทึกเป็น pending ต้องกดอนุมัติจึงส่ง LINE
  • ให้ AI วิเคราะห์เพิ่ม — AI จะประเมิน Confidence และ R:R ก่อนบันทึก
  • Confidence ขั้นต่ำ — ถ้า AI ให้คะแนนต่ำกว่าค่านี้ ระบบจะข้ามสัญญาณ

💬 LINE Official Account

1. สร้าง OA ใหม่ (ถ้าโควต้าเก่าหมด)

  1. เข้า LINE Developers Console
  2. สร้าง Provider และ Messaging API channel
  3. ไปที่แท็บ Basic settings คัดลอก Channel Secret
  4. ไปที่แท็บ Messaging API กด Issue Channel Access Token
  5. เปิด Allow bot to join group chats เป็น Enabled
  6. ใส่ Webhook URL: https://forex-rouge-gamma.vercel.app/api/line/webhook

2. เอา Group ID

  1. เชิญ OA เข้ากลุ่ม
  2. ให้มีคนส่งข้อความในกลุ่ม 1 ครั้ง
  3. เปิด URL: https://forex-rouge-gamma.vercel.app/api/line/group-id
  4. เอา Group ID ไปใส่ใน MT5 Signal Settings
⚠️ ระวัง: ถ้าเปลี่ยน OA ใหม่ ต้องใช้ Group ID ใหม่เสมอ แม้จะเป็นกลุ่มเดียวกันก็ตาม

📁 ติดตั้ง EA ใน MT5

  1. ดาวน์โหลดไฟล์ mt5/BOS_LuxAlgo_Bridge.mq5 จาก GitHub
  2. คัดลอกไปที่โฟลเดอร์ MQL5/Experts/ ของ MT5
  3. เปิด MetaEditor
  4. กด Compile (F7)
💡 Tip: ถ้า compile ไม่ผ่าน ตรวจสอบว่า LuxAlgo SMC indicator ได้ติดตั้งใน chart แล้ว

✅ ทดสอบระบบ

วิธีตรวจสอบว่าทำงาน
  1. รอให้มี BOS หรือ CHoCH ใหม่บน chart
  2. ดู log MT5 ว่าขึ้น [Bridge] ✅ SIGNAL CREATED!
  3. เข้า Admin → สัญญาณเทรด ดูว่ามีสัญญาณใหม่
  4. ตรวจสอบกลุ่ม LINE ว่าได้รับข้อความหรือไม่

หรือทดสอบผ่านหน้า MT5 Signal Settings → ทดสอบส่ง LINE ได้เลย

🔍 Troubleshooting

Authentication failed

LINE Access Token ผิด หรือยังไม่ได้ redeploy หลังแก้ env

  • ตรวจสอบ token ว่าถูกต้องและไม่มีช่องว่าง
  • กด Redeploy ใน Vercel
  • ตรวจสอบว่า token ยังไม่หมดอายุ

You have reached your monthly limit

โควต้า LINE หมดแล้ว ต้องสร้าง OA ใหม่หรืออัปเกรดแพลน

Failed to send messages

บอทยังไม่ได้อยู่ในกลุ่ม หรือ Group ID ผิด

  • เชิญ OA เข้ากลุ่ม
  • เปิด Allow bot to join group chats
  • เอา Group ID ใหม่จาก /api/line/group-id

Signal ไม่ขึ้นใน Admin

MT5 ยังไม่ส่งถึง backend

  • ตรวจสอบ API_URL ใน EA
  • ตรวจสอบ API_KEY ว่าตรงกับ Vercel
  • ตรวจสอบว่าเปิด Allow WebRequest ใน MT5 แล้ว

📊 โควต้า LINE

แพลนฟรีของ LINE Messaging API มีโควต้า 300 ข้อความ/เดือน

ตรวจสอบโควต้าคงเหลือได้ที่หน้า Admin → MT5 Signal Settings → โควต้า LINE คงเหลือ

💡 ประหยัดโควต้า:
  • เปิด Admin อนุมัติก่อนส่ง LINE
  • ปิด AI analysis ถ้าไม่จำเป็น
  • ไม่กดทดสอบบ่อยเกินไป

❓ FAQ

Q: ทำไมต้องสร้าง OA ใหม่?

ถ้าโควต้า 300 ข้อความ/เดือนหมด จะไม่สามารถส่งได้จนกว่าจะรีเซ็ต (1 ของทุกเดือน) หรืออัปเกรดแพลน

Q: สามารถส่งไปหลายกลุ่มพร้อมกันได้ไหม?

ได้ ให้เพิ่มเป้าหมายใน MT5 Signal Settings แล้วติ๊กใช้งานทุกเป้าหมาย

Q: ถ้าปิด AI analysis จะเกิดอะไรขึ้น?

Signal จะถูกบันทึกทันทีโดยไม่ตรวจสอบ Confidence และข้อความ LINE จะใช้ template แบบเดิม

Q: Admin อนุมัติคืออะไร?

Signal จาก MT5 จะบันทึกเป็น pending ต้องไปที่หน้า Admin → สัญญาณเทรด แล้วกด อนุมัติ จึงจะส่ง LINE
